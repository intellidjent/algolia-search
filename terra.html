What's new in WorkFusion 10.0 Terra

The first WorkFusion release in the Terra series addresses the current system architecture shortcomings. The most noticeable improvement is replacing several databases with a single one to resolve a variety of storage-related issues.

Bot Orchestration Platform

To address the current monolithic architecture that is strongly tied to all other components of Control Tower, we introduce a brand new state-of-the-art Bot Orchestration Platform.

With improved process orchestration and bot isolation the new platform enables true horizontal scalability, improved capacity planning and more control over bots.

Unified Database

WorkFusion v10.0 introduces Unified DB based on MS SQL. A single database with the following benefits: Reliability and Monitoring, Tuning, Support, Maintenance, and Troubleshooting.

Smarter Bots

In SPA Terra, we continue to focus on our main advantage — AutoML. This release is a leap in AI performance with AutoML 10 accelerating Time-to-Value, improving our No Code Machine Learning and Automated Intelligent Training.
