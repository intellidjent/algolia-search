SPA Intro

Components Architecture

Application level

Control Tower

Control Tower is the central managing web application, which orchestrates automation processes and tasks, and provides a user interface for power users, administrators, and developers. Provides functionality to manage Business Processes, Manual Tasks, AutoML automations, access to Analytics, Bot Manager, user management system, and Secrets Vault.

Related components

  • WorkSpace
  • WorkflowRPA
  • AutoML
  • OCR
  • Data Storage

Covered functionality

  • Automation processes management
  • Creation and editing workflows using GUI tools
  • Running and monitoring automations
  • Input/output data handling
  • User/role management
  • Manual Tasks management
  • Workforce management
  • System-wide configuration

WorkSpace

WorkSpace is the web application for manual task fulfillment by Subject Matter Experts.

Related components

  • Control Tower
  • Object Storage

Covered functionality

  • Manual tasks queue management
  • Manual task fulfillment by Subject Matter Experts
  • Managing users, roles, teams, qualifications

Bot Manager

Bot Manager is the web application that uses Bot Agents on RPA servers to manage RPA units and send commands.

Related components

  • Control Tower
  • RPA
  • Analytics
  • Database
  • Secrets Vault

Covered functionality

  • Server processes management
  • Maintenance of active RDP sessions
  • Hardware utilization monitoring
  • Statistics for bots and instances

Service level

AutoML

AutoML finds the best ML model for a particular target function (expressed in a Manual Task), trains it, validates, and finally predicts the target variables.

Related components

  • Control Tower
  • Data Storage

Covered functionality

  • Search for the best model within hypermodels space
  • Training, validation of models using data from Manual Tasks
  • Computing predictions for unseen input data
  • Visualization of prediction quality

BEP

Bot Execution Platform (BEP) is the engine that increases performance of AutoML and amount of running workers. The platform creates tasks and submits them to queues.

Related components

  • Control Tower
  • AutoML
  • Workflow
  • RPA

Covered functionality

  • Horizontal scaling of workers
  • Bot execution isolation
  • Reuse of the shared resources (cluster nodes) both by CT and AutoML tasks
  • Task processing fault-tolerance
  • Aggregation of logs and metrics from tasks and workers

RPA

RPA Cluster is a scalable computing environment that executes user scenarios within desktop or web applications. It takes control of the keyboard and mouse pointer to emulate user inputs.

Related components

  • Control Tower
  • Bot Manager
  • BEP
  • Secrets Vault
  • Database

Covered functionality

Automation of user scenarios within desktop and web applications.

OCR

Optical Character Recognition component (OCR) of WorkFusion SPA digitizes image-based data into character documents (for example, transforms a PDF file into TXT).

Related components

  • Control Tower
  • WorkSpace
  • Object Storage
  • Secrets Vault
  • Database
  • RabbitMQ

Covered functionality

OCR processing of scanned documents.

Workflow

Workflow Engine executes automation processes in Control Tower. It orchestrates work queues, Machine Learning training data collection, exception handling, and quality control.

Related components

  • Control Tower
  • RPA
  • WorkSpace
  • AutoML

Covered functionality

Automation Process execution according to process definition created in Control Tower. Execution is handled in different computing environments—RPA, WorkSpace, OCR, and AutoML.

Analytics

Analytics web application provides comprehensive dashboards and reporting capabilities.

Related components

  • Control Tower
  • Bot Manager

Covered functionality

  • Reporting
  • Dashboards

Data level

Database

Database stores business process definitions, runtime and tracking information on their executions—including RPA and Cognitive automations.

Related components

  • Control Tower
  • Bot Manager
  • WorkSpace
  • Analytics
  • OCR

Covered functionality

Reliable storage of the designed automation implementation.

Secrets Vault

Secrets Vault is a secured storage destination for sensitive data.

Related components

  • Control Tower
  • Bot Manager
  • RPA
  • OCR

Covered functionality

This is mostly related to the solution’s non-functional requirements:

  • Storage of authentication data (login/password) for applications involved in RPA and providing secure access to it during business process execution.
  • Storage of authentication data for WorkFusion SPA inter-component interaction.

Object Storage

Object Storage keeps binary file type content, such as OCR results, AutoML models, development packages (artifacts), screenshots, and reports generation by automation.

Related components

  • Workflow
  • OCR
  • AutoML

Covered functionality

Reliable storage of OCR results and trained AutoML models.
