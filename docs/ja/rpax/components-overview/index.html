Components Overview

WorkFusionは、幅広いコンポーネントを提供し、包括的な エンタープライズオートメーション。それぞれの目的とコンポーネントの目的を理解する 最初に一緒に仕事をしてみましょう。

次に、各コンポーネントの説明とその機能について説明します。

用途

ワークスペース

WorkSpaceは、Subject Matter Expertによる手動タスクの実行のためのWebアプリケーションです。

関連コンポーネント：コントロールタワー、オブジェクトストレージ

対象機能：

  • 手動タスクのキュー管理。
  • 主題専門家による手作業の完了。
  • ユーザー、役割、チーム、資格の管理。

管制塔

Control Towerは、中央管理Webアプリケーションであり、自動化プロセスやタスクを編成し、パワーユーザー、管理者、および開発者向けのユーザーインターフェイスを提供します。

関連するコンポーネント：WorkSpace、Workflow、RPA、AutoML、OCR、データストレージ

対象機能：

  • 自動化プロセス管理
  • GUIツールを使用したワークフローの作成と編集
  • 自動化の実行と監視
  • 入力/出力データ処理
  • ユーザーと役割の管理
  • 手動タスク管理
  • 労働力管理
  • システム全体の構成

サービス

AutoML

注：SPA経由でのみご利用いただけます

AutoMLは、特定のターゲット関数（Manual Taskで表現されている）のための最良のMLモデルを見つけ、トレーニングし、検証し、最終的にターゲットの変数を予測します。

関連コンポーネント：コントロールタワー、データストレージ

WorkFusion SPAにはAutoMLというネイティブ機能があり、データクレンジング、モデル選択、モデルトレーニングをデータ科学者が手動で手動で自動化します。このソフトウェアには、MLモデルが作業中のパターンを識別できるように、ビジネス・ユーザーが非構造化データにタグを付けるために使用するインターフェースが含まれています。作業パターンの事例が十分に見られると、ソフトウェアは人間またはより高い信頼水準で作業の実行方法を予測します。 AutoMLを使用すると、ビジネスユーザにスケーラブルなセルフサービス機能を提供することで、AIの配備と拡張に要する時間とコストを大幅に削減できます。

AutoMLは、Wordドキュメント、PDF、TIFFだけでなく、スプレッドシート、電子メール、テキストメッセージ、Webページ、およびコメントフィールドからも学習できます。一般的なクラウドベースのAIプロバイダのような何百万ものサンプルを必要とするのではなく、300〜500個のサンプルを最適にトレーニングしてから、画期的な自動化率を実現するという点で、SPAはAI分野ではユニークです。追加サンプルが1つ増えるごとに、自動化率が増加します。

ビジネス全体の多くの自動化ユースケースには同じタスクがあります。たとえば、銀行業務では、モーゲージ文書とSWIFTメッセージは異なる形式のデータで、異なるプロセスで使用されますが、どちらも人の名前と住所が含まれています。 WorkFusion SPAのネイティブ機能であるTransfer Learningは、あるユースケースの学習を同じタスクの別のユースケースに適用します。これにより、MLの学習と導入に要する時間が短縮されます。

RPA

RPAクラスタは、デスクトップまたはWebアプリケーション内のユーザーシナリオを実行するスケーラブルなコンピューティング環境です。ユーザ入力をエミュレートするためにキーボードとマウスポインタを制御します。

RPAの主な目標は、デスクトップおよびWebアプリケーション内のユーザーシナリオの自動化です。

関連コンポーネント：Control Tower |

OCR

OCRエンジンはスキャンされたドキュメントを処理し、非構造化イメージベースのデータをいくつかのフォーマット（html、xml）の構造化文字ドキュメントに変換します。

関連コンポーネント：Control Tower、WorkSpace、データストレージ

ワークフロー

Workflow Engineは、Control Towerで作成された自動化プロセスを実行します。これはControl Tower内で設定されたサーバーアプリケーションです。

Automation Control Towerで作成されたプロセス定義に従ってプロセスを実行します。実行はさまざまなコンピューティング環境（RPA、WorkSpace、OCR、AutoML）で処理されます。

関連コンポーネント：Control Tower、RPA、WorkSpace、AutoML

Analytics

注：SPAのみで利用可能

Analytics Webアプリケーションは、包括的なダッシュボードとレポート機能を提供します。

関連コンポーネント：Control Tower、WorkSpace

WorkFusion Analytics Dashboardsは、メトリックと主要業績評価指標（KPI）を集約して表示するレポート作成メカニズムです。ダッシュボードは、コンテキスト内の洞察を明らかにし、ビジネスパフォーマンスに伝達することによって意思決定を改善するのに役立ちます。

ダッシュボードは、1つ以上の図表、表、および数値メトリックで構成されています。各SPAセットアップでは、すぐに使用できるダッシュボードのパッケージが提供されます。 WorkFusion SPAチームまたはWorkFusionパートナーは、カスタムセットのオンデマンドダッシュボードを提供して、オートメーション固有の結果を可能な限り最良の方法で視覚化することもできます。

プラットフォームモニタ

プラットフォームモニタは、SPAプラットフォームコンポーネントの集中監視を担当します。

関連するコンポーネント：

プラットフォームモニタ

プラットフォームモニタは、SPAプラットフォームコンポーネントの集中監視を担当します。

関連コンポーネント：Control Tower、WorkSpace、RPA、OCR、Analytics

対象機能：

  • 全体的なプラットフォームコンポーネントの状態を監視する。
  • ハードウェアの利用状況と状態を監視する。
  • 問題に関する警告。
  • サードパーティのアプリケーション監視ツールと統合するためのAPIを提供する。

Secrets Vault

Secrets Vaultは機密データの保護された格納先です。

関連コンポーネント：コントロールタワー、RPA

これは、ほとんどの場合、ソリューションの非機能要件に関連しています。

  • RPAに関係するアプリケーションの認証データ（ログイン/パスワード）を格納し、ビジネスプロセスの実行中にRPAに安全なアクセスを提供する。
  • WorkFusion SPAのコンポーネント間の相互作用に関する認証データの保存。

データベース

データベースには、ビジネスプロセス定義、ランタイム、RPAや認知自動化などの実行に関する情報が格納されます。これは、設計されたオートメーション実装の信頼できるストレージです。

関連コンポーネント：Control Tower、WorkSpace、Analytics

オブジェクトストレージ

Object Storageは、OCR結果、AutoMLモデル、スクリーンショット、レポート生成などのバイナリファイルタイプのコンテンツを自動化して保持します。これは、OCRの結果と訓練されたAutoMLモデルの信頼できるストレージです。

関連するコンポーネント：ワークフロー、OCR、AutoML
