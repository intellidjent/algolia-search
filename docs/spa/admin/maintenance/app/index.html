APP

Php-fpm (TinyMCE)

  • Management: wfmanager {start | stop | restart | status} php-fpm
  • Log files: <install-dir>/supervisord/log/php-fpm.log
  • Startup scripts:
    • <install-dir>/supervisord/apps/php-fpm.ini
    • <install-dir>/php/etc/php.ini
  • Configuration files:
    • Service directory: <install-dir>/php/
    • General configuration: <install-dir>/php/etc/php-fpm.conf
  • Default ports: unix socket: {{ php_home }}/php5-fpm.sock

Bot Manager

  • Management: wfmanager {start | stop | restart | status} bot-manager
  • Log files: <install-dir>/supervisord/log/bot-manager.log
  • Startup scripts:
    • <install-dir>/supervisord/apps/bot-manager.ini
    • <install-dir>/bot-manager/run-bot-manager.sh
  • Configuration files:
    • Service directory: <install-dir>/bot-manager/
    • General bot-manager configuration: <install-dir>/bot-manager/config/
  • Default ports: tcp 5555

  • Configuration files:
