Install Linux servers

Overview

note

Before you proceed, make sure, you have prepared the servers according to Prepare for Installation and executed Precheck.

The SPA component must be installed in the following order:

  1. MS SQL server
  2. INT server
  3. BEP master server
  4. BEP agent(s) server(s)
  5. APP server
  6. OCR server (Linux)

All installation steps must only be run from the <package-dir> directory on your Integration server.

cd <package-dir>

Install

MS SQL Server

  1. On the INT server, check that MS SQL meets the requirements:

    $ ./install.sh precheck mssql

  2. If all checks have passed successfully, run the following command to grant necessary permissions for database schemes.

    $ ./install.sh configure mssql

INT Server

To install the Integration Server:

  1. As a root user, run the following command:

    ./install.sh preinstall int

  2. As a <wf_user>, run the following commands:

    $ ./install.sh install int
$ ./install.sh check int

BEP Master Server

To install the BEP Master Server:

  1. As a root user, run the following command:

    $ ./install.sh preinstall bep-master

  2. As a <wf_user>, run the following commands:

    $ ./install.sh install bep-master
$ ./install.sh check bep-master

BEP Agent Server

To install the Agent Server:

  1. As a root user, run the following command:

    $ ./install.sh preinstall bep-agent

  2. As a <wf_user>, run the following commands:

    $ ./install.sh install bep-agent
$ ./install.sh check bep-agent

APP Server

To install the APP Server:

  1. As a root user, run the following command:

    $ ./install.sh preinstall app

  2. As <wf_user>, run the following commands:

    $ ./install.sh install app
$ ./install.sh check app

OCR Server (Linux)

To install the OCR Linux Server:

  1. As a root user, run the following command:

    $ ./install.sh preinstall ocr

  2. As a <wf_user>, run the following commands:

    $ ./install.sh install ocr
$ ./install.sh check ocr
