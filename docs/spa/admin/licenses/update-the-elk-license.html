Update ELK license

Initial installation of WorkFusion SPA version 10.0 enables the 30-day trial license for ELK. For production purposes, it is required to update the license in Kibana.

important

The WorkFusion ELK License file must be provided by the installation team or your account manager.

To update the ELK license:

  1. Go to Kibana, for example, at https://app_hostname/kibana.

  2. Authenticate to Kubana, using the ELK admin credentials that you specify in the config.yml file during the installation.

  3. On the left menu, click Management.

  4. Go to the license management section.

  5. Click Update license.

  6. Select or drag your license file to the window. The license is uploaded to Kibana and updated automatically.
